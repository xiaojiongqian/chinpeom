# 古代学级称号多语言翻译方案

## 项目概述

本方案针对唐诗猜测应用中的学级系统，恢复了被清理的等级说明与多语言翻译细节，遵循翻译“信、雅、达”的基本原则，为 6 种语言提供了一致的文化表达。

## 翻译原则与方法

### “信、雅、达”原则应用

1. **信（忠实）**：准确传达原有的等级制度与学术地位含义。
2. **雅（优美）**：选择各语言中优雅、正式的学术术语。
3. **达（通达）**：确保不同语言背景的使用者能理解等级层次。

### 翻译策略

- **英语**：借鉴历史学衔体系；
- **德语**：参考传统大学学位与荣誉体系；
- **西班牙语**：结合本地学术传统；
- **法语**：采用 Académie 体系中的正式称谓；
- **日语**：保留汉字原貌，突出文化传承。

## 翻译设计理念

### 英语
- **Commoner / Student Scholar**：突出成长路径。
- **First Degree / Stipend / Tribute Scholar**：借用传统学衔。
- **Third/Second/Top Place Graduate**：对应殿试排名。

### 德语
- **Bakkalaureus、Magister、Lizentiat、Doktor**：沿用德语学位称谓。
- **Summa/Magna cum Laude**：指示荣誉等级。
- **Universalgelehrter**：体现终极学者地位。

### 西班牙语
- **Plebeyo**：强调社会阶层起点。
- **Bachiller / Licenciado / Doctor en Letras**：与现代学位对应。
- **Laureado**：表现登榜荣誉。

### 法语
- **Roturier**：历史平民称谓。
- **Bachelier ès Lettres / Agrégé / Docteur ès Lettres**：经典学术等级。
- **Lauréat 系列**：突出殿试名次。

### 日语
- 采用汉字原词，避免歧义；
- 兼顾日语对中国科举的既有认知；
- 保持与中文文案一一对应。

## 技术实现要点

1. **数据来源**：`src/types/index.ts` 中的等级配置保持分数段与翻译键一致。
2. **国际化键名**：在 `src/locales/*` 中以 `rankDescriptions.<rank>` 存储，可用于弹窗与成就页。
3. **组件展示**：`AchievementView` 与 `RankDialog`（若存在）按当前语言读取映射；中文模式优先使用上表“中文说明”。
4. **扩展性**：新增语言时，参照本表追加列并同步 i18n 资源。

## 文化与教育价值

- 以游戏化方式传递科举制度层级；
- 帮助国际用户理解中国传统学术称号；
- 在界面中提供文化注释，提升沉浸感。

## 维护建议

1. 文案调整需同步更新表格与各语言包；
2. 若增减学级，需同步更新 `src/types/index.ts`、成就视图与本说明；
3. 建议在 PR 模板中注明是否改动学级配置，以便快速回溯。
